PING Architecture

BeAPI gives you the ability to 'ping' all the api servers in your architecture to test for uptime. To configure this, add all servers to your 'beapi_api.yml' file under secondaryServer:

apitoolkit:
    ...
    secondaryServer: ['http://localhost:8080']

    ...

Once all server have been added properly, you can ping all server for uptime through a simple api call:​

curl -v -H "Content-Type: application/json" -H "Authorization: Bearer "  --request GET "http://localhost:8080/v1.0/server/pingServers/"
