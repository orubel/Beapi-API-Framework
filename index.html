Installation

Getting started with BeAPI is simple process. Below are the steps for installation and configuration of your Beapi API Automation Tool Suite...


Requirements

The BeAPI™ API Framework has the following project requirements:

* Grails 3.3.0 (or higher)
* Java 1.8 (or higher)

Step 1: Installation

Type the following command to create a new project with the Beapi Framework Installed:

grails create-app myapp --profile org.grails.profiles:beapi:1.0.0

Step 2: Make Configuration Directories

Create a file in your home directory for your configuration files using the following commands in a shell:

mkdir ~/.beapi
cd ~/.beapi
mkdir ~/.beapi/.iostate
  • Move beapi.yml, beapi_api.yml,beapi_db.yml, beapi_server.yml, beapi.groovy
    • from 'grails-app/conf' into '/.beapi/'
  • Now move all files from 'src/iostate' into '/.beapi/.iostate'

Step 3: Configure

Next follow these configuration steps to get your environment setup and working:

  • Configure Server
  • Configure Database
  • Configure API

Step 4: Build & Test

You can now build the project using the following command:

./gradlew --stop;./gradlew clean;./gradlew build --stacktrace  --refresh-dependencies

Step 5: Run

If you configured it properly, there should be no errors and you can now run the project using the following command:

java -jar build/libs/beapi-backend-0.1.jar

Step 6: Test

You can now test out your running project using the following command (replacing 'login' and 'password' with root.login and password found in your beapi.yml file)::

curl -v -H "Content-Type: application/json" -X POST -d '{"username":"login","password":"password"}' http://localhost:8080/api/login

If everything ran right, you should see output similar to this:

{"username":"admin","authorities":["ROLE_ADMIN"],"token_type":"Bearer","access_token":"q3s2ov25h8jll757ff5baqhf8nscbjf9"}
